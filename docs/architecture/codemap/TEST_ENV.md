## Running Tests

```bash
# Full suite (CI style)
uv run pytest tests/ -v --cov=atman --cov-fail-under=90

# Fast (skip slow tests)
uv run pytest tests/ -m "not slow" -v

# Integration only
uv run pytest tests/ -m "integration" -v

# Single file
uv run pytest tests/path/to/test_foo.py -v
```
